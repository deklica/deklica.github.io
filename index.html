Enable ZIP installation in Kodi: - Open Kodi > Settings > System > Add-ons > Enable "Unknown sources"

Method 1: Install via Repository ZIP file
- Download dEXE's Addons Repository
- In Kodi, go to: Add-ons > Install from ZIP file
- Navigate to and select the downloaded ZIP
- Go to Install from repository and select Addon
- Install preferred Addon
- Autoupdate will work

Method 2: Install via Kodi File Manager (direct GitHub source)
- Open Kodi > go to Settings > File Manager
- Click Add source
- Use the following URL: https://deklica.github.io/
- Name it something like dEXE's Addons and press OK
- Now go to Add-ons > Install from ZIP file
- Select the source (dEXE's Addons) and install the repository ZIP
  or You can also just install your preferred addon.
- Then go to Install from repository > dEXE's Addons Repository
- Install preferred Addon
- Autoupdate will work (If you installed the addon through a repository)
